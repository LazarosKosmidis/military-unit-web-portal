# Military Unit Web Portal

Full-stack web application developed during military service for internal organizational management and communication.

## Features

- Centralized access to internal services
- Contact directory
- Real-time chat
- User management
- Internal resource navigation

## Technologies

- React
- Node.js
- Express
- Socket.io
- JavaScript

## Purpose

Designed to simplify access to local services and improve communication workflows within the organization.
